# 신뢰 기반 음식점 리뷰 분석(restaurant_review_analysis)

## 1. 프로젝트 개요

본 저장소는 성균관대학교 데이터사이언스융합전공 `데이터마이닝실습(DSC 3009)` 최종 프로젝트의 분석 코드와 재현성 설명을 담고 있다.

본 프로젝트는 네이버 지도와 카카오맵 음식점 리뷰를 대상으로, 광고성 또는 저신뢰 리뷰를 필터링하고 고신뢰 리뷰의 감성 점수를 기반으로 음식점 평가 지표를 산출하는 것을 목표로 한다. 단순 평균 별점은 플랫폼별 리뷰 작성 문화, 리뷰 이벤트, 짧고 일반적인 칭찬 리뷰의 영향을 받을 수 있으므로, 본 프로젝트에서는 리뷰 신뢰도와 텍스트 감성을 함께 반영한 보정 지표를 설계하였다.

## 2. 분석 목적

본 프로젝트의 주요 목적은 다음과 같다.

1. 네이버 지도와 카카오맵 리뷰에서 광고성 또는 저신뢰 리뷰를 식별한다.
2. 리뷰 길이, 사진 여부, 방문 인증, 메뉴명, 숫자 표현, 경험 표현, 재방문 표현 등 다양한 feature를 활용하여 trust score를 계산한다.
3. 수동 라벨링 데이터를 기준으로 신뢰도 필터링 모델의 성능을 평가한다.
4. 고신뢰 리뷰를 대상으로 음식점 리뷰 도메인에 맞춘 감성분석을 수행한다.
5. 카카오맵의 실제 별점을 참고하여 감성 점수를 1~5점 척도의 감성 별점으로 정규화한다.
6. 최종적으로 플랫폼별 리뷰 성향 차이와 신뢰 기반 음식점 평가 지표의 의미를 해석한다.

## 3. 데이터 설명

분석 데이터는 네이버 지도와 카카오맵에서 수집한 음식점 리뷰로 구성되어 있다.

- 분석 지역: 서울 성수, 마포
- 음식점 유형: 한식 음식점
- 음식점 수: 20개
- 수집 기간: 2025-01-01 ~ 2026-05-09
- 전체 리뷰 수: 약 6,700건
  - 네이버 지도: 약 4,175건
  - 카카오맵: 약 2,526건

주요 변수는 다음과 같다.

| 변수명 | 설명 |
|---|---|
| `platform` | 리뷰 플랫폼 |
| `store_name` | 음식점명 |
| `account_id` | 리뷰 작성자 식별값 |
| `account_review_count` | 작성자의 전체 리뷰 수 |
| `visit_date` | 방문일 또는 리뷰 기준 날짜 |
| `verification_method` | 방문 인증 방식 |
| `review_text` | 리뷰 본문 |
| `review_length` | 리뷰 길이 |
| `has_photo` | 사진 첨부 여부 |
| `visit_count` | 네이버 방문 횟수 |
| `rating` | 카카오 실제 별점 |
| `account_avg_rating` | 카카오 작성자 평균 별점 |
| `reviewer_level` | 카카오 리뷰어 등급 |

## 4. 데이터 공개 범위 및 익명화

원본 리뷰 데이터에는 플랫폼 계정명, 계정 식별값, 리뷰 원문 등 민감할 수 있는 정보가 포함되어 있다. 따라서 본 공개 저장소에는 전체 원본 데이터를 포함하지 않는다.
